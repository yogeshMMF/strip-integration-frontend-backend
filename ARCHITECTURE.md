## Key Components

### 1. Frontend (Angular 20)

**Files:**

- `payment.component.ts` - Main payment logic
- `payment.component.html` - UI template
- `payment.component.css` - Styling
- `payment.service.ts` - API communication
- `environment.ts` - Configuration

**Responsibilities:**

- Display payment form
- Collect customer information
- Mount Stripe Card Element
- Handle payment confirmation
- Display success/error messages

### 2. Backend (Node.js/Express)

**Files:**

- `server.js` - Express server with Stripe integration

**Responsibilities:**

- Create Payment Intents
- Validate requests
- Handle webhooks
- Confirm payments
- Process refunds

### 3. Stripe Integration

**Components:**

- **Stripe.js** - Frontend library for secure card collection
- **Stripe API** - Backend API for payment processing
- **Webhooks** - Asynchronous event notifications
